# UniTime-Plateforme-Intelligente-de-Gestion-des-Emplois-du-Temps
UniTime est une application web complète développée pour la Faculté des Sciences et Techniques de Tanger (FSTT). Elle automatise la gestion des emplois du temps universitaires grâce à un moteur algorithmique sans conflits, des mises à jour en temps réel, et un assistant IA intégré appelé NEON (basé sur Llama 3).
